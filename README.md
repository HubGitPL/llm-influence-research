# Wpływ agentów LLM na rozwój projektów open-source
## Opis projektu
Celem projektu jest zbadanie, w jaki sposób narzędzia oparte na LLM-ach, takie jak Codex czy Claude Code, a także autonomiczne agenty, takie jak OpenClaw, wpływają na rozwój projektów open-source.

Generowanie kodu stało się łatwe i szybkie. Może to prowadzić do przyspieszenia rozwoju projektów open-source (szybsze wdrażanie nowych funkcjonalności, krótszy czas iteracji, sprawniejsze przeglądy i akceptacja pull requestów). Może jednak również doprowadzić do ich osłabienia, ponieważ tworzenie kodu we własnym, zamkniętym projekcie może stać się prostsze niż wnoszenie wkładu do istniejących bibliotek lub frameworków.

Projekt będzie polegał na monitorowaniu wybranych projektów open-source oraz porównaniu ich aktualnych metryk z danymi historycznymi. Przykładowe metryki do analizy:

liczba i częstotliwość publikacji nowych wersji pakietów (np. w npm, PyPI, Maven Central)
liczba commitów i zgłoszeń (issue) w repozytoriach GitHub
liczba nowych funkcjonalności i poprawek błędów (na podstawie informacji w release notes)
liczba aktywnych kontrybutorów
czas od zgłoszenia problemu do jego zamknięcia.
